    # Byte structure:
    0: Command length (8 bytes)
    |    1: Execute immediately (always 0)
    |    |    2: Command code (Port Output)
    |    |    |    3: Port
    |    |    |    |    4: Mode
    |    |    |    |    |    5: Sub-command (WriteDirectModeData)
    |    |    |    |    |    |    6: Payload checksum (?)
    |    |    |    |    |    |    |    7: Value (100)
    |    |    |    |    |    |    |    |
